# Bank Account Management System

A simple Bank Account Management project built using **HTML, CSS, and JavaScript**.

This project simulates basic banking operations such as deposits, withdrawals, checking balance, and viewing transaction history.

---

## Features

- Deposit money
- Withdraw money
- Check current balance
- View all deposits
- View all withdrawals
- Transaction history stored in an array

---

## Technologies Used

- HTML
- CSS
- JavaScript (ES6 Classes)
