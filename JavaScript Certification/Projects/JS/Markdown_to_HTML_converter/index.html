📘 Markdown cheatsheet

  • # H1 → # Heading 1
  • ## H2 → ## Heading 2
  • ### H3 → ### Heading 3
  • **bold** → bold
  • *italic* → italic
  • [text](URL) → link
  • ![alt](src) → image
  • > quote → blockquote

📋 Example

Input markdown:
