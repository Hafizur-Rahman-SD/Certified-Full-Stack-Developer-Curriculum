# 🎹 Keyboard Music Machine

An interactive **Keyboard Music Machine** built with **HTML, CSS, and JavaScript**.
This project allows users to play drum sounds using their **computer keyboard**.

Unlike a traditional drum pad that only works with a few keys, this version allows **almost any keyboard key to trigger a sound**, creating a more **fun and dynamic music experience**.

---

# 🚀 Live Features

✅ Play sounds using keyboard keys
✅ Random drum beat generation
✅ Real-time key display
✅ Clean and simple UI
✅ Fast and responsive sound playback

---

# 🛠 Built With

* HTML5
* CSS3
* JavaScript (Vanilla JS)

# 🎮 How To Use

1. Open the project in your browser.
2. Press **any key on your keyboard**.
3. A drum sound will play instantly.
4. The pressed key will appear on the screen.

Enjoy making beats with your keyboard! 🎵
